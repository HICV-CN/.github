# HiCV | 嗨简历

[![Website](https://img.shields.io/badge/官网-hicv.cn-2f6fd6)](https://hicv.cn)
[![Resume Templates](https://img.shields.io/badge/简历模板-resume--templates-4c7bd1)](https://github.com/HICV-CN/resume-templates)
[![Resume Guide](https://img.shields.io/badge/写作指南-resume--guide-16a34a)](https://github.com/HICV-CN/resume-guide)

嗨简历是面向中文求职者的 AI 简历制作与求职准备平台，围绕简历模板、AI 简历生成、简历优化、岗位适配、求职攻略、校招信息和面经复盘，帮助用户更快完成一份可投递的简历。

> 找模板、写内容、改版本、准备投递，都可以从 [hicv.cn](https://hicv.cn) 开始。

## 核心资源

| 仓库 | 内容 | 适合谁 | 入口 |
| --- | --- | --- | --- |
| [resume-templates](https://github.com/HICV-CN/resume-templates) | 嗨简历模板库：应届生、实习、社招、留学、英文、岗位专项模板 | 需要快速选择简历版式并在线编辑的求职者 | [浏览模板](https://hicv.cn/templates) |
| [resume-guide](https://github.com/HICV-CN/resume-guide) | 简历写作建议：ATS 优化、经历表达、岗位匹配、投递前检查 | 第一次写简历、换岗、校招、社招用户 | [阅读攻略](https://hicv.cn/blog) |

## 为什么整理到 GitHub

- 让简历模板和写作建议有一个公开、稳定、便于收藏的资源入口。
- 方便用户从 GitHub 直接跳转到 hicv.cn 在线使用模板。
- 把“模板怎么选”和“内容怎么写”拆成两个仓库，降低查找成本。
- 后续可以持续补充岗位模板、写作清单和求职准备方法。

## 在 hicv.cn 继续完成简历

- [免费简历模板](https://hicv.cn/templates) - 在线浏览、筛选、套用模板。
- [求职攻略](https://hicv.cn/blog) - 学习简历写作、AI 优化、面试与投递方法。
- [简历范文](https://hicv.cn/fanwen) - 参考不同岗位和经历类型的表达方式。
- [校招信息](https://hicv.cn/xiaozhao) - 跟进校招节点、岗位和投递机会。
- [面经复盘](https://hicv.cn/mianjing) - 面试前查看公司和岗位相关经验。

## 适用场景

- 应届生、实习生第一次写简历。
- 社招用户准备跳槽、转岗或晋升版本。
- 留学、英文简历、行业岗位专项模板选择。
- 投递前进行 ATS 关键词、经历成果和岗位匹配检查。
